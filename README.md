

Welcome to the **K-Pop Idol Stats Dashboard**!  
This web app lets you explore statistics of your favorite K-Pop idols, such as followers, albums, songs, and awards â€” all displayed visually with charts.

---

## Features

- Select your favorite K-Pop idol from a dropdown menu
- View detailed stats including:
  - Number of followers
  - Albums released
  - Songs performed
  - Awards won
- Interactive chart visualization powered by Chart.js
- Responsive and user-friendly design
---

## How to Run Locally

1. Clone this repository or download the ZIP file.
2. Open the project folder.
3. Open `index.html` in your preferred web browser.  
   *(No server or Node.js required)*
4. Interact with the dashboard directly!
